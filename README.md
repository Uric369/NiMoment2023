# Features
- 展示年度业务数据
- 展示个人业务数据
- 展示个人年度成就
- 实现动态UI
- 优化用户交互体验

## UI设计与交互体验
- 首页
   - 跟随鼠标视差滚动
   - 呼吸文字效果，打字机动画，同时加载数据
- 部门数据界面
   - 飞船降落视差滚动
   - message进入视口后延迟弹出
   - 查看消息
- 个人数据界面
   - 动态UI（组件呼吸、旋转效果）
   - 数字攀升计数
   - 昼夜切换，白天/黑夜响应按钮
- 个人成就界面
   - 视觉冲击卡片
